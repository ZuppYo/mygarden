# สร้างใบเสนอราคาตามราคากลาง (เทียบใบเสนอราคาจริง)

## Task Requirement

- Goal: สร้างใบเสนอราคาอ้างอิง (benchmark quotation) ที่ใช้รายการและจำนวนเดียวกับใบเสนอราคาจริง แต่คิดราคาตามช่วงราคากลางตลาด เพื่อให้เปรียบเทียบกับใบเสนอราคาจากบ้านสวนธรรมรัตน์ได้ชัดเจน
- In scope: ดึงรายการ/จำนวนจาก [007-analyze-landscaping-quotation](007-analyze-landscaping-quotation.md) และ [quotation.pdf](../resources/quotation/quotation.pdf), กำหนดสเปกวัสดุมาตรฐานที่สมเหตุสมผล, คำนวณราคา/หน่วยและราคารวมตามราคากลาง, จัดทำใบเสนอราคา benchmark เป็น HTML และตารางเทียบกับใบเสนอราคาจริง (เปิดดูในเบราว์เซอร์ได้)
- Out of scope: ต่อรองราคากับผู้รับเหมา, สั่งจ้างงานจริง, แก้ไขใบเสนอราคาจริง
- Assumptions: ใช้จำนวนพื้นที่/แผ่นตามที่ระบุในใบเสนอราคาจริง (หญ้า 10 ตร.ม., หินเกล็ดดำ 210 ตร.ม., ปูพื้นภายนอก 34 ตร.ม., คอนกรีต 54 ตร.ม., แผ่นทางเดิน 34 แผ่น) จนกว่าจะยืนยันจำนวนที่ถูกต้องกับผู้รับเหมา

## Checklist

- [x] T001 [N] สรุปรายการ จำนวน และสเปกวัสดุมาตรฐานที่ใช้คำนวณราคากลาง จากผลวิเคราะห์ใน [007-analyze-landscaping-quotation](007-analyze-landscaping-quotation.md)
  - ✅ ใช้จำนวนตามใบเสนอราคาจริง และกำหนดสเปกมาตรฐานสำหรับคำนวณราคากลางใน HTML
- [x] T002 [N] กำหนดราคา/หน่วยราคากลางแต่ละรายการ (หญ้า, หินเกล็ดดำ, ปูพื้นภายนอก, เทคอนกรีต, แผ่นทางเดินทรายล้าง) พร้อมอ้างอิงแหล่งข้อมูลตลาด
  - ✅ กำหนดราคากลาง: หญ้า 120 บาท/ตร.ม., หินเกล็ดดำ 300 บาท/ตร.ม., ปูพื้นภายนอก 750 บาท/ตร.ม., คอนกรีต 450 บาท/ตร.ม., แผ่นทรายล้าง 180 บาท/แผ่น
- [x] T003 [N] คำนวณราคารวม benchmark และยอดมัดจำ 50% ให้สอดคล้องรูปแบบใบเสนอราคาจริง
  - ✅ ยอดรวมราคากลาง 120,120 บาท; มัดจำ 50% 60,060 บาท; ยอดคงเหลือ 60,060 บาท
- [x] T004 [N] สร้างใบเสนอราคา benchmark เป็น HTML ที่ `resources/quotation/benchmark-quotation.html` (รูปแบบอ่านง่าย พิมพ์ได้)
  - ✅ สร้าง [benchmark-quotation.html](../resources/quotation/benchmark-quotation.html) พร้อม CSS สำหรับอ่านและพิมพ์
- [x] T005 [N] สร้างตารางเปรียบเทียบ benchmark vs ใบเสนอราคาจริง (ราคา/หน่วย, ราคารวม, ส่วนต่าง %) ใน HTML เดียวกันหรือไฟล์ `resources/quotation/benchmark-comparison.html`
  - ✅ รวมตารางเปรียบเทียบใน HTML เดียวกัน; ใบเสนอราคาจริงสูงกว่าราคากลาง 58,630 บาท หรือประมาณ 48.8%
- [x] T006 [N] สรุปข้อสังเกตและช่วงราคาที่ยังต้องยืนยันสเปกก่อนตัดสินใจจ้าง
  - ✅ เพิ่มข้อควรยืนยันก่อนใช้เปรียบเทียบจริงและช่วงราคากลางที่ควรใช้ต่อรองใน HTML

## Output

- HTML quotation: [benchmark-quotation.html](../resources/quotation/benchmark-quotation.html)
- Benchmark total: 120,120 บาท
- Benchmark deposit 50%: 60,060 บาท
- Actual vendor quote: 178,750 บาท
- Difference: actual quote is higher by 58,630 บาท (+48.8%)

## Pricing Assumptions

| รายการ | จำนวน | ราคากลาง/หน่วย | ราคารวม |
|---|---:|---:|---:|
| หญ้า | 10 ตร.ม. | 120 บาท/ตร.ม. | 1,200 บาท |
| หินเกล็ดดำ | 210 ตร.ม. | 300 บาท/ตร.ม. | 63,000 บาท |
| งานปูพื้นภายนอก | 34 ตร.ม. | 750 บาท/ตร.ม. | 25,500 บาท |
| งานเทคอนกรีตหน้าบ้าน | 54 ตร.ม. | 450 บาท/ตร.ม. | 24,300 บาท |
| แผ่นทางเดินทรายล้าง 40x80 ซม. | 34 แผ่น | 180 บาท/แผ่น | 6,120 บาท |

Note: ราคานี้เป็น benchmark เพื่อเปรียบเทียบ ไม่ใช่ราคาจ้างจริง และควรยืนยันสเปกวัสดุ/หน้างานกับผู้รับเหมาก่อนใช้ต่อรอง

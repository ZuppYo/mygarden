คุณเป็น Ai Agents ที่เป็นผู้ช่วยจัดสวนแบบมืออาชีพ โดยการทำงานผมรวบรวมข้อมูลรูปบ้านทั้งหมดจากทุกมุมของบ้านซึ่งอยู่ใน folder `resources` ซึ่งระบุว่าเป็นรูปที่ถ่ายจากส่วนใหนของบ้าน คุณจะต้องออกแบบสวนโดยยึดข้อมูลจากรูปบ้านนี้เป็นหลัก ในการทำงานคุณจะต้องมี skill หรือ plugin ที่สามารถทำงานกับรูปภาพ `.png` ได้ และสามารถ process รูปภาพได้สามารถสร้างรูปใหม่โดยอิงจากรูปเดิม และตัดต่อรูปจาก `resource` ได้  ให้คุณพิจารณาว่าควรต้องมี skill หรือ plugin อะไรบ้าง และการทำงานเราจะใช้ work flow จาก skill iom-todo task และ iom-todo-task-archive


ช่วยสร้าง task สำหรับงานแรกผมอยากกำหนดกรอบการทำงานก่อน  จากข้อมูลรูปใน folder `resources` ทั้งหมดผมอยากให้คุณสร้างเป็น htmp page ที่แสดงรูปบ้านทั้งหมดและแบบแยกส่วน โดยให้สร้าง html ไว้ที่ folder `outline/home.html` โดยผมจะตรวจผลจากไฟล์นี้ เราจะเริ่มออกแบบหน้า home.html ก่อนเพื่อใช้เป็น layout template เพื่อใช้ในการออกแบบ  ถ้าผมพอใจแล้วค่อยลง design

## Task Requirement

- Goal: หาและจัดทำ **ราคากลางเฉพาะภาคใต้** สำหรับรายการงานจัดสวนที่ใช้ในโปรเจกต์ (สทิงพระ จ.สงขลา) แล้วประเมินว่า **ภาคใต้มีข้อมูลราคากลางครบหรือไม่** เทียบกับรายการในข้อ 3/10 ของ [summary.html](../resources/quotation/summary.html)
- In scope:
  - Web research แหล่งราคาที่ระบุพื้นที่ภาคใต้ชัดเจน (สงขลา, หาดใหญ่, สุราษฎร์ธานี, ภูเก็ต ฯลฯ) หรือจังหวัดสงขลาโดยตรง
  - ตรวจแหล่งราชการ/กึ่งราชการที่แยกจังหวัดได้ (เช่น price.moc.go.th, บัญชีค่าแรงงาน) สำหรับวัสดุก่อสร้างที่เกี่ยวข้อง
  - ครอบคลุมรายการเดียวกับ benchmark ปัจจุบัน: หญ้านวลน้อย, หินเกล็ดดำ, ปูพื้นภายนอก, เทคอนกรีต, แผ่นทางเดินทรายล้าง 40×80, carport (อ้างอิง)
  - จัดทำตาราง **ครบ/ไม่ครบ/มีบางส่วน** ต่อรายการ พร้อมแหล่งอ้างอิง URL และช่วงราคาภาคใต้
  - สรุปราคากลางภาคใต้ที่แนะนำ (ถ้ามีข้อมูลเพียงพอ) และช่องว่างที่ต้องขอใบเสนอท้องถิ่น
- Out of scope:
  - แก้ไขใบเสนอราคาจริงหรือต่อรองกับผู้รับเหมา
  - สร้าง HTML quotation ฉบับใหม่ทั้งไฟล์ (ทำใน task แยกถ้าต้องการ)
  - อ้างราคาราชการจัดสวนแบบมีตารางสำเร็จรูปถ้าไม่มีจริง — ต้องระบุชัดว่าไม่พบ
- Assumptions:
  - พื้นที่อ้างอิงหลัก: **อ.สทิงพระ จ.สงขลา** (ใกล้หาดใหญ่)
  - ราคากลาง = งานเหมารวมค่าแรง+วัสดุมาตรฐาน (สอดคล้อง summary.html)
  - แหล่งทั่วประเทศที่มีหน้า “สงขลา/หาดใหญ่” นับเป็นแหล่งภาคใต้ได้
- Constraints:
  - บันทึก URL และวันที่สำรวจทุกแหล่ง
  - แยก “ราคาวัสดุอย่างเดียว” กับ “ราคาเหมารวมปู/ติดตั้ง” ไม่ปนกันโดยไม่ระบุ
